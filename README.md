Drop-in caching for OpenAI, Anthropic, and any LLM API. Tracks cost savings. Enables offline testing.

```js
const cache = createCache({ dir: '.llm-cache', verbose: true });
const openai = cache.wrap(new OpenAI()); // that's it
```
